### Установка

#### Клонируйте репозиторий:

`git clone https://github.com/G28XYZ/form-react.git`

#### Перейдите в папку склонированного репозитория

`cd form-react`

#### Установите npm зависимости из package.json:

`npm i`

#### Запустите приложение:

`npm run dev`
